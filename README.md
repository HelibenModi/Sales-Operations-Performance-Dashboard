# 📊 Sales & Operations Performance Dashboard (Power BI)

## 📌 Overview
This project is an end-to-end Business Intelligence solution built using Power BI. It analyzes sales, product, and customer data to generate actionable business insights.

---

## ⚙️ Tools Used
- Power BI
- Power Query (ETL)
- DAX
- Excel / CSV

---

## 📊 Key Features
- Interactive dashboards with KPIs (Sales, Profit, Margin)
- Data modeling using relational tables
- DAX calculations for business metrics
- Filters, slicers, and drill-down analysis

---

## 📈 Key Insights
- Top products contribute ~40% of total revenue
- Ontario region generates highest sales
- Some products have low profit margins

---

## 📂 Project Structure
- data/ → raw datasets
- dashboard/ → Power BI file
- docs/ → screenshots and model

---

## 🚀 How to Run
1. Open Power BI Desktop
2. Load CSV files from /data
3. Open .pbix file
