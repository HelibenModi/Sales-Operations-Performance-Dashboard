# Sales Data Analysis Dashboard (Power BI)

##  Project Overview

This project analyzes over 10,000+ sales records to identify key business insights related to revenue, profit, customer segments, and regional performance. The solution includes interactive dashboards built in Power BI with a structured data model and DAX-based calculations.

##  Objectives

* Analyze sales performance across products and regions
* Identify top-performing and low-performing products
* Track revenue trends over time
* Improve business decision-making using data visualization

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Excel / CSV (Data Sources)
* Data Modeling (Star Schema)

##  Data Model

* Fact Table: `sales`
* Dimension Tables: `products`, `customers`, `date`
* Relationships: One-to-Many (*:1)
* Implemented star schema for optimized performance

## Key Features

* Interactive dashboards
* KPI tracking (Revenue, Profit, Profit Margin)
* Top N product analysis
* Time-based trend analysis using date table
* Region and customer segmentation insights

## Key Insights

* Top 10 products contribute ~35–45% of total revenue
* Certain products show high sales but low profit margins (<20%)
* Kitchen category performs best in revenue generation
* Opportunities identified for optimizing low-margin products


## How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Interact with filters and visuals
